# 공지사항 정책

### 공지사항

- 관리자, 회장, 임원만 `등록`할 수 있다.
- 등록한 사람 혹은 등록한 사람보다 권한이 높은 사람만 `수정`할 수 있다.
- 등록한 사람 혹은 등록한 사람보다 권한이 높은 사람만 `삭제`할 수 있다.
- 비회원을 제외한 모든 사용자가 `조회`할 수 있다.

### 공지사항 댓글

- 등록한 사람 혹은 등록한 사람보다 권한이 높은 사람만 `삭제`할 수 있다.
